# Hourglass Accessibility Conformance

Hourglass is moving towards compliance with WCAG 2.2 AA.  More details can be found in the [Hourglass pane of the Accessiblity Project](https://github.com/orgs/openlibhums/projects/49/views/14).

## Results Key

| Result | Markdown Symbol |
|---|---|
| Not applicable | :brown_square: `:brown_square:` |
| Partially supports | :orange_circle: `:orange_circle:` |
| Supports | :white_check_mark: `:white_check_mark:` |
| Does not support | :x: `:x:` |
